# ChatGPT Wrapper 'Writing Assistant'

## Setup
1. Get API key from OpenAI api
2. `OPENAI_API_KEY="your-key"`
3. Run View.java

## Features
- [x] Formal, casual writing mode
- [x] Save/load version history

## Design Patterns
- Singleton: ChatGPT wrapper class is declared as singleton
- Observer: UI updates
- Factory: Version control that constructs Version objects for each iteration
